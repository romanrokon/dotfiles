# Server / SBC apt manifest. Installed per-package (tolerant of missing
# packages — see _apt_install_tolerant in 03-apps-bg.sh), so it's fine if a
# package isn't in this distro's repos.
#
# Notes on cross-distro availability:
#   eza      — Debian 13+ / Ubuntu 24.04+ (not in Debian 12 main). Install via
#              cargo or upstream .deb if missing — exa is the older fork.
#   lazygit  — not in Debian main; Ubuntu PPA or Go install required.
#   bat      — binary is 'batcat' on Debian/Ubuntu (aliased in zsh).
#   fd-find  — binary is 'fdfind' on Debian/Ubuntu (aliased in zsh).
